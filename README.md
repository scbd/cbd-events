## Build Setup


# install dependencies
``` bash
$ yarn install
```

# serve with hot reload at localhost:3000
``` bash
$ yarn dev
```

# build ios platform for production
``` bash
$ yarn build:ios
```

# build android platform for production
``` bash
$ yarn build:android
```
