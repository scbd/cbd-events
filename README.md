# CBD-Events PWA



## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate

# to build cordova ISO wrapper
$ yarn cordova

# to open the project xcode and the simulator on OSX
$ yarn ios:sim
```
